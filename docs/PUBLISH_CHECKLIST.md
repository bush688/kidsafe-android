# 发布检查清单

## 应用内

- [ ] “帮助/关于”页面包含版权与联系方式
- [ ] 版本号与应用名称正确（与商店一致）
- [ ] 全功能回归：无明显崩溃、无明显 UI 破版、滚动正常

## Google Play

- [ ] 图标、功能图、截图准备完成（见 `docs/play-store/ASSETS.md`）
- [ ] 上架文案准备完成（见 `docs/play-store/LISTING.md`）
- [ ] 应用分类与内容分级配置完成
- [ ] 定价与分发地区配置完成
- [ ] 生成 AAB（Release + 已签名）并上传

## GitHub

- [ ] README / LICENSE / CHANGELOG 完整
- [ ] Release 附件准备（APK/AAB/校验值）
- [ ] 仓库清理：移除机器相关文件（如 `local.properties`）、避免提交 keystore
