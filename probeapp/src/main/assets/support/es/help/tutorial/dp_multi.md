# Calibración multipunto (nivel DP)

Úsalo cuando ambos transmisores estén instalados igual y el error sea aproximadamente lineal.

Registra varios puntos estables:

- Nivel de referencia %% (p. ej., A)
- Presión diferencial actual de B (ΔPnow)
- LRV/URV actuales de B

Luego abre **Nivel DP → Recalcular rango → Multipunto** y calcula el nuevo LRV/URV.
