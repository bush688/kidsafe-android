# 差压液位计多点校正

适用前提：两台变送器安装方式一致，介质与工况一致，误差可近似为线性（零点偏移 + 跨度偏差）。

## 输入准备

在多个稳定液位点上，记录：

- 参考液位%%（例如 A 的显示值或实测值）
- B 的当前差压 ΔPnow（建议直接读取 B 的差压值）
- B 当前配置的 LRV/URV（原量程）

## 操作步骤

1. 进入“差压液位计计算 → 量程反算 → 多点校正”
2. 选择“拟合跨度”（推荐）或“固定跨度”
3. 添加并填写多个点（至少 2 点，建议 ≥3 点）
4. 点击“计算”，得到新的 LRV/URV 与误差指标

## 如何判定结果是否可信

- RMSE 越小，代表拟合越好
- 最大误差越小，代表各点一致性越好
- 若误差明显偏大，建议检查数据点是否误录、或工况是否不一致
